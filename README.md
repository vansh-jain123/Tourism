# 🌍 Smart AI Tourism Recommendation System

A modern AI-powered tourism web application that helps users discover the best travel destinations across India based on their preferences such as budget, travel type, and interests.

Built using React + Vite, this project focuses on performance, scalability, and a smooth user experience.

---

## 🚀 Features

- 🤖 AI-based destination recommendations
- 📍 Explore popular tourist destinations across India
- 🧭 Travel categories:
  - Solo trips
  - Couple trips
  - Friends / Group trips
- 🔍 Smart search and filtering system
- 🖼️ Dynamic destination images using APIs
- 📊 Interactive and responsive dashboard UI
- 🌐 Real-time data using external APIs

---

## 🛠️ Tech Stack

- Frontend: React.js (Vite)
- Styling: CSS / Tailwind CSS
- State Management: Zustand (optional)
- Routing: React Router
- APIs:
  - Location API
  - Image API (Unsplash / Pexels)

## ⚙️ Installation & Setup

### 1. Clone the Repository
git clone https://github.com/your-username/smart-tourism.git
cd smart-tourism

### 2. Install Dependencies
npm install

### 3. Start Development Server
npm run dev

### 4. Build for Production
npm run build

---

## 🔑 Environment Variables

Create a `.env` file in the root directory:

VITE_IMAGE_API_KEY=your_api_key  
VITE_LOCATION_API_URL=your_api_url
